# Project Title: Ransomware Detection Through Behavioral Indicators - Analysis of the Colonial Pipeline Attack

## Introduction

This project focuses on understanding how behavioral indicators can be used to detect ransomware attacks at an early stage. Using the **2021 Colonial Pipeline ransomware incident** as our case study, we analyze how the attack unfolded, why the IT–OT interdependence created widespread disruption, and how improved behavioral detection methods can strengthen critical infrastructure security.

## Aim of the Project

To study the Colonial Pipeline ransomware attack and enhance **Malware & Attack Simulation** techniques by identifying, refining, and validating **behavioral indicators** that can detect ransomware before it impacts critical systems.

## Expected Outcome

* A clear assessment of IT–OT risks exposed during the Colonial Pipeline attack.
* Evaluation of response effectiveness, containment actions, and intelligence measures.
* Identification of key behavioral indicators that could improve early ransomware detection.
* Policy-level insights for strengthening cybersecurity across critical infrastructure.
